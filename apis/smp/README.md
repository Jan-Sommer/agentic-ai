## Common Patterns Across All Waves Products

| Pattern | Detail |
|---------|--------|
| **Framework** | Symfony 7.4 + API Platform 3.4.17 (all PHP backends) |
| **Auth** | JWT Bearer — shared public key (`JWT_PUBLIC_KEY` env) |
| **Roles** | `ROLE_SMP_ADMIN > ROLE_COMPANY_ADMIN > ROLE_{PRODUCT}_ADMIN > ROLE_{PRODUCT}_USER > ROLE_{PRODUCT}_READ_ONLY` |
| **File storage** | Azure Blob Storage (most), AWS S3 (tendering) |
| **Database** | PostgreSQL + Doctrine ORM |
| **OpenAPI** | Auto-generated by API Platform at `/api/docs.json` (dev) |
| **Health** | `GET /api/health` — public, all backends |
| **Export formats** | PDF (mPDF), Excel (PhpSpreadsheet), CSV, JSON, ZIP |
